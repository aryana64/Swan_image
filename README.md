# Swan_image
Кейс Минприроды РФ по детекции подвидов лебедей на изображениях
Хакатон "Цифровой прорыв, сезон ИИ"
В рамках программы по неинвазивному мониторингу состояния окружающей среды в рамках учёта редких животных необходимо разработать автономное программное решение по классификации птиц на данных фотосъёмки. Классы: лебедь малый, лебедь-кликун, лебедь-шипун.
